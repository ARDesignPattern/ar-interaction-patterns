Technology Overview

This module presents the technical implementation of AR interaction design patterns using Unity, tailored for HoloLens 2 and developed with AR Foundation and Mixed Reality Toolkit (MRTK) 2.8.3.

All components were created and tested in Unity 2020.3.25f1. The base project includes a well-structured directory with Unity's Assets/, Packages/, and ProjectSettings/ folders, optimized for cultural heritage applications.

To get started, users can follow the setup guide in the GitHub repository to install the necessary MRTK packages—either via unitypackage files or using the Mixed Reality Feature Tool. Once the environment is prepared, you can download and import specific design pattern prefabs as .unitypackage files from the GitHub Releases section.

Each Unity prefab corresponds to a specific AR interaction pattern and encapsulates its interaction logic for immediate reuse. These prefabs are modular and field-tested, enabling easy integration into your own HoloLens-based AR applications.
